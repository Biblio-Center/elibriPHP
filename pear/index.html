eLibri's PEAR channel

Registering the channel: 
pear channel-discover elibri.com.pl/system/pear
Listing available packages: 
pear remote-list -c elibri
Installing a package: 
pear install elibri/package_name
Installing a specific version/stability: 
pear install elibri/package_name-1.0.0
pear install elibri/package_name-beta
Receiving updates via a feed: 
https://www.elibri.com.pl/system/pear/feed.xml
